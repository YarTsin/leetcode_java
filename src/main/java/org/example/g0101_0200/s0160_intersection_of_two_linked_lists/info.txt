Пример
List A:      a1 → a2
                   ↘
                     c1 → c2 → c3
                   ↗
List B: b1 → b2 → b3

Пример работы
Возьмем пример из начала:
List A: a1 → a2 → c1 → c2 → c3 (длина 5)
List B: b1 → b2 → b3 → c1 → c2 → c3 (длина 6)

Указатели пройдут:
pointerA: a1, a2, c1, c2, c3, b1, b2, b3, c1
pointerB: b1, b2, b3, c1, c2, c3, a1, a2, c1

Они встретятся в c1.
