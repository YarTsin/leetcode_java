Определение: Минимальная глубина бинарного дерева
- это количество узлов вдоль кратчайшего пути от корневого
узла до ближайшего листового узла.
Важно: Листовой узел - это узел без детей.

Сравнение решений
Рекурсивное DFS (O(N)):
Временная сложность: O(N)
Пространственная сложность: O(H) для стека вызовов
Простое, но может быть неэффективно для несбалансированных деревьев

BFS с очередью (ОПТИМАЛЬНОЕ для минимальной глубины):
Временная сложность: O(N) в худшем случае, но обычно быстрее
Пространственная сложность: O(N)
Находит минимальную глубину сразу при первом листе

DFS со стеком (O(N)):
Временная сложность: O(N)
Пространственная сложность: O(H)
Должен обойти все узлы чтобы найти минимальную глубину
