Сравнение решений
Рекурсивное DFS (оптимальное):
Временная сложность: O(N)
Пространственная сложность: O(H) для стека вызовов
Простое и элегантное

BFS с очередью:
Временная сложность: O(N)
Пространственная сложность: O(N)
Обходит дерево уровень за уровнем

DFS со стеком:
Временная сложность: O(N)
Пространственная сложность: O(H)
Имитирует рекурсию с помощью стека
