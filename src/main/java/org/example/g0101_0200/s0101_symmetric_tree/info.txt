Сравнение решений
Рекурсивное DFS (оптимальное):
Временная сложность: O(N)
Пространственная сложность: O(H) для стека вызовов
Наиболее понятное и элегантное

Итеративное BFS (очередь):
Временная сложность: O(N)
Пространственная сложность: O(N) для очереди
Избегает переполнения стека

Итеративное DFS (стек):
Временная сложность: O(N)
Пространственная сложность: O(H) для стека
Аналог рекурсивного без рекурсии
