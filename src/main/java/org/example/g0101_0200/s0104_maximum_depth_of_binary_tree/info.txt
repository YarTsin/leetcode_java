Определение: Максимальная глубина бинарного дерева -
это количество узлов вдоль самого длинного пути от корневого узла
до самого дальнего листового узла.

Сравнение решений
Рекурсивное DFS (оптимальное):
Временная сложность: O(N) - посещаем каждый узел один раз
Пространственная сложность: O(H) для стека вызовов (H - высота дерева)
Простое и элегантное

Итеративное BFS (очередь):
Временная сложность: O(N)
Пространственная сложность: O(N) для хранения узлов уровня
Находит глубину по уровням

Итеративное DFS (стек):
Временная сложность: O(N)
Пространственная сложность: O(H) для стека
Явно отслеживает глубину каждого узла

Особенности задачи
Важное уточнение: Глубина измеряется количеством узлов, а не ребер. Поэтому:
Пустое дерево: глубина 0
Только корень: глубина 1
Корень + один ребенок: глубина 2
