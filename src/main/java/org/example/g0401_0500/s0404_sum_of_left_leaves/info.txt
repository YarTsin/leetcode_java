Оптимальное решение (Рекурсивное с передачей информации о положении)
Объяснение подхода:
Рекурсивно обходим дерево, передавая информацию о том,
является ли текущий узел левым потомком
Проверяем, является ли узел листом и левым потомком
Суммируем результаты из левого и правого поддеревьев
Временная сложность: O(n) - каждый узел посещается один раз
Пространственная сложность: O(h) - высота дерева


Рекомендация: Рекурсивное решение с передачей параметра
isLeftChild является наиболее понятным и точно определяет
левые листья. Для больших деревьев лучше использовать
итеративное решение чтобы избежать переполнения стека.
