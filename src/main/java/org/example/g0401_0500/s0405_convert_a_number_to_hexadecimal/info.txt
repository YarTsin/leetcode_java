Особенности обработки отрицательных чисел
Ключевые моменты:

Используем беззнаковый сдвиг (>>>), чтобы корректно обрабатывать отрицательные числа

Для отрицательных чисел автоматически работает дополнение до двух:

-1 → 0xffffffff

-2 → 0xfffffffe

Ведущие нули не добавляются благодаря условию num != 0

==============

Сравнение подходов
Первый подход:
Более интуитивный
Может сделать лишние итерации для больших чисел

Второй подход:
Фиксированное число итераций (максимум 8)
Более предсказуемая производительность
Чуть более оптимальный
