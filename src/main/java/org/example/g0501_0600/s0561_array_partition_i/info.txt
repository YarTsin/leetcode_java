Оптимальное решение с сортировкой массива и выбором каждого второго элемента:
Просто в реализации
Эффективно для большинства случаев
Легко понимается и объясняется

Альтернативные решения могут быть полезны в специфических случаях:
Counting sort эффективен при ограниченном диапазоне значений
Решения с частичной сортировкой могут быть быстрее для очень больших массивов
Для большинства практических случаев стандартное
решение с сортировкой является лучшим выбором.
