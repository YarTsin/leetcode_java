Решение: математическое - через площадь треугольника
Основной подход:
Три точки образуют бумеранг если площадь образованного ими треугольника
не равна нулю. Площадь треугольника можно вычислить по формуле
через координаты вершин:

Площадь = 0.5 * |(x1(y2 - y3) + x2(y3 - y1) + x3(y1 - y2))|

Если площадь равна 0, точки коллинеарны (лежат на одной прямой).
Также нужно проверить, что все точки различны.

Упрощенная проверка:
Так как нас интересует только факт нулевой площади,
можно использовать упрощенную версию без абсолютного
значения и деления на 2:

(x1 - x2) * (y2 - y3) != (x2 - x3) * (y1 - y2)

Это эквивалентно проверке наклона между парами точек.

========
Все три решения основаны на одной математической концепции -
проверке коллинеарности точек. Первое решение (через площадь треугольника)
является наиболее прямолинейным и понятным. Все решения имеют O(1) сложность
по времени и памяти, так как работают с фиксированным количеством точек.
