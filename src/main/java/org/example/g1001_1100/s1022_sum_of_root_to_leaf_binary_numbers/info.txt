Решение - рекурсивный DFS с накоплением значения

Основной подход:
Используем обход дерева в глубину (DFS). При спуске по дереву
мы накапливаем текущее значение числа, сдвигая уже накопленное
значение на один бит влево и добавляя значение текущего узла.

Формула для обновления значения на каждом шаге:
current_value = (current_value << 1) | node.val

Это эквивалентно:
current_value = current_value * 2 + node.val
Когда мы достигаем листового узла (у которого нет потомков),
добавляем накопленное значение к общей сумме.
