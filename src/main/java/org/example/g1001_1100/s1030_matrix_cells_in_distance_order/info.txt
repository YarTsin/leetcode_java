Решение - с использованием BFS и очереди

Используем обход в ширину (BFS) от центральной точки.
Этот подход естественным образом дает ячейки в порядке
возрастания расстояния, так как BFS сначала посещает все
ячейки на расстоянии 0, затем 1, затем 2 и т.д.

Алгоритм:
Создаем матрицу для отслеживания посещенных ячеек
Используем очередь для BFS
Начинаем с центральной ячейки, добавляем ее в очередь и отмечаем как посещенную
На каждом шаге извлекаем ячейку из очереди, добавляем ее в результат
Добавляем в очередь все непосещенные соседние ячейки (вверх, вниз, влево, вправо)
Повторяем пока очередь не пуста
