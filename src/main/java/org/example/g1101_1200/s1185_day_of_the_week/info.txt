Решение - Использование формулы Зеллера
Подход:

Используем формулу Зеллера для вычисления дня недели:
h = (q + ⌊13(m+1)/5⌋ + K + ⌊K/4⌋ + ⌊J/4⌋ - 2J) mod 7

где:
h - день недели (0 = Saturday, 1 = Sunday, 2 = Monday, ..., 6 = Friday)
q - день месяца
m - месяц (3 = March, 4 = April, ..., 12 = December, 13 = January, 14 = February)
K - год века (year % 100)
J - нулевое столетие (floor(year / 100))
