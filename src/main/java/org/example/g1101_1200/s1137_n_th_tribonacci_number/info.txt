Подходы к решению
1. Рекурсивное решение (наивное)
Прямая рекурсивная реализация, но неэффективная.

2. Динамическое программирование (с мемоизацией)
Хранение вычисленных значений для повторного использования.

3. Итеративное решение (оптимальное)
Последовательное вычисление чисел в цикле.

4. Матричное возведение в степень (продвинутое)
Для O(log n) времени выполнения.

=================

Временная сложность
O(n) - выполняем n-2 итерации цикла (для n > 2),
каждая итерация выполняется за O(1).

Пространственная сложность
O(1) - используем только константное количество переменных.
