Объяснение подходов
1. Метод с префиксными/суффиксными массивами (оптимальный)
Префиксный массив: Хранит информацию о том, является ли начало
массива строго возрастающим
Суффиксный массив: Хранит информацию о том, является ли конец
массива строго возрастающим
Проверка соединения: Для подмассива [i,j] проверяем,
что левая часть (0..i-1) и правая часть (j+1..n-1) обе строго
возрастающие и что последний элемент левой части меньше первого
элемента правой части


2. Прямая проверка (простой подход)
Перебор всех подмассивов: Проверяем каждый возможный подмассив
Проверка валидности: Для каждого подмассива проверяем, является
ли оставшийся массив строго возрастающим
Недостаток: Медленный для больших массивов из-за тройной вложенности

Ключевые моменты решения
Пустой массив считается строго возрастающим
Подмассив должен быть contiguous (последовательным)
Строгое возрастание означает, что каждый следующий элемент
должен быть строго больше предыдущего

Эффективность достигается за счет предварительного
вычисления префиксных и суффиксных массивов
