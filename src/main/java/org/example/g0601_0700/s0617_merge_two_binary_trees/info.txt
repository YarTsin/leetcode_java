Сравнение подходов
Рекурсивный подход - наиболее простой и читаемый, хорошо работает
для сбалансированных деревьев

Итеративный со стеком - избегает потенциального переполнения
стека для очень больших деревьев

Итеративный с очередью - обход в ширину, может быть полезен
для определенных типов деревьев
