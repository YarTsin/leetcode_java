Сравнение подходов
BFS с очередью - более интуитивно понятен для задачи обхода по уровням, использует дополнительную память для очереди
DFS с рекурсией - более элегантен, но требует хранения дополнительных списков для сумм и количеств, может привести к переполнению стека для очень больших деревьев

Рекомендация: BFS подход предпочтительнее, так как он:
Более прямолинеен для задачи обхода по уровням
Избегает потенциального переполнения стека
Легче читается и понимается
Имеет такую же временную сложность O(n)
