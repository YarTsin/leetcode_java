Объяснение подхода
Почему ответ всегда 0, 1 или 2:
0: Для пустой строки
1: Если строка уже палиндром - удаляем всю строку за один шаг
2: Для любой непалиндромной строки:
Первый шаг: удаляем все экземпляры одного символа (например, все 'a')
Второй шаг: удаляем все экземпляры другого символа (все 'b')

Доказательство для случая 2:
Все символы 'a' образуют палиндромную подпоследовательность
(так как все одинаковы)
Все символы 'b' также образуют палиндромную подпоследовательность
После удаления всех 'a' остаются только 'b', которые тоже палиндромны
