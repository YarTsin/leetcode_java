Сравнение подходов
Итеративный подход: Понятен для новичков
Временная сложность: O(log n) для каждой итерации (в худшем случае несколько итераций)
Пространственная сложность: O(1)

Математический подход:
Оптимальный по времени (O(1))
Требует знания математического свойства
Используется в производственном коде для эффективности
