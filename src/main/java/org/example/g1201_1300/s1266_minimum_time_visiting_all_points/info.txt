Математический подход на основе Chebyshev distance

Основная идея: Для перемещения между двумя точками A(x1, y1) и B(x2, y2)
минимальное время равно максимуму из абсолютных разностей
по координатам - это известно как расстояние Чебышева.

Почему это работает:
Диагональное движение позволяет одновременно изменять обе координаты
Минимальное время определяется большей из двух разностей:

dx = |x2 - x1| (разность по x)
dy = |y2 - y1| (разность по y)
Если dx > dy: сначала двигаемся по диагонали dy раз, затем по горизонтали dx - dy раз
Если dy > dx: сначала двигаемся по диагонали dx раз, затем по вертикали dy - dx раз
В любом случае общее время: max(dx, dy)

Подход:
Инициализировать totalTime = 0
Для каждой пары последовательных точек (от i=0 до points.length-2):
Вычислить разности координат: dx = |xᵢ₊₁ - xᵢ|, dy = |yᵢ₊₁ - yᵢ|
Добавить к общему времени max(dx, dy)
Вернуть totalTime
