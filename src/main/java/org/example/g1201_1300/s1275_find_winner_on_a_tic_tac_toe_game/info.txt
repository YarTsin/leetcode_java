Решение - Отслеживание сумм строк, столбцов и диагоналей

Подход:
Создаем массивы для отслеживания:
rows[3] - суммы для строк
cols[3] - суммы для столбцов
diag1 - сумма для главной диагонали
diag2 - сумма для побочной диагонали

Присваиваем значения:
Игрок A: +1
Игрок B: -1

После каждого хода проверяем, не достигнута ли выигрышная комбинация:
Если какая-то строка/столбец/диагональ равна +3 → победа A
Если какая-то строка/столбец/диагональ равна -3 → победа B

После всех ходов:
Если есть победитель - возвращаем его
Если нет победителя и ходов было 9 - возвращаем "Draw"
Иначе - возвращаем "Pending"
