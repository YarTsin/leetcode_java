Решение - Однопроходный алгоритм с накоплением результата

Основная идея: Проходим по списку один раз, на каждом шаге
сдвигаем текущий результат на 1 бит влево и добавляем текущий бит.

Подход:
Инициализировать result = 0
Пройти по всем узлам списка:
Сдвинуть текущий результат на 1 бит влево: result = result << 1
Добавить текущий бит: result = result | current.val
Вернуть result

Математическое обоснование:
Каждый сдвиг влево эквивалентен умножению на 2
Операция OR с битом добавляет его к числу
