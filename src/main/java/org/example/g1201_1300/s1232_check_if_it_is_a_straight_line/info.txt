Использование векторного произведения

Математическая основа:
Три точки (x₁,y₁), (x₂,y₂), (x₃,y₃) лежат на одной прямой, если:
(x₂ - x₁) * (y₃ - y₁) = (y₂ - y₁) * (x₃ - x₁)

Шаги:
Проверка количества точек - если точек ≤ 2, они всегда на прямой
Вычисление эталонных разностей - по первым двум точкам
Проверка коллинеарности - для каждой последующей точки
проверяем условие через cross product
Ранний выход - при обнаружении точки не на прямой сразу возвращаем false

Преимущества:
Избегает проблем с делением на ноль
Не использует числа с плавающей точкой
