Решение - с Hashmap

Создать карту (HashMap), которая будет сопоставлять первый элемент
каждого кусочка с самим кусочком. Это ключевая идея, так как первый
элемент каждого кусочка уникален (благодаря уникальности всех чисел)
и служит идеальным ключом для быстрого поиска.

Пройти по целевому массиву arr и для каждого элемента:

Если этот элемент является началом какого-либо кусочка
(т.е., присутствует в карте как ключ), взять соответствующий
кусочек и проверить, совпадает ли последовательность элементов
этого кусочка с последующей последовательностью в arr.

Если все кусочки совпали последовательно и полностью, вернуть true.
Если где-то несовпадение — false.
