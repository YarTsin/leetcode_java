Решение - с использованием скользящего окна

Основной подход
Обработать случай k = 0 - сразу вернуть массив нулей.
Создать расширенный массив длиной 2 * n, чтобы удобно работать
с круговой природой данных.
Использовать технику скользящего окна для эффективного вычисления сумм.
Вычислить начальную сумму для первого окна.
Сдвигать окно и обновлять сумму, вычитая уходящий элемент и добавляя приходящий.
