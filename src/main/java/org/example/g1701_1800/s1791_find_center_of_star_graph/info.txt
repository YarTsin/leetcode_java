Решение - использование свойства центра звезды

Решением является использование свойства центра звезды:
центр должен присутствовать в каждом ребре.

Подход:
Достаточно рассмотреть только первые два ребра из массива
Центр звезды должен быть вершиной, которая присутствует в обоих ребрах
Сравниваем вершины первых двух ребер и находим общую вершину
