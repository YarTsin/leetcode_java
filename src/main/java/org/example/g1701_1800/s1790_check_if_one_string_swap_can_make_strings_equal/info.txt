Решение - с подсчетом позиций

Решением является подсчет позиций, где символы различаются,
и проверка определенных условий.
Подход:
Проверить, что строки имеют одинаковую длину
Найти все индексы, где символы в строках различаются
Если различий нет - строки уже равны, возвращаем true
Если различий ровно 2 - проверяем, можно ли обменом сделать строки равными
Если различий больше 2 - невозможно исправить одним обменом
