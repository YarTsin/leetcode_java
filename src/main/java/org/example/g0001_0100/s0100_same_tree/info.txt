Сравнение решений
Рекурсивное (оптимальное):
Временная сложность: O(N) - посещаем каждый узел один раз
Пространственная сложность: O(H) для стека вызовов (H - высота дерева)
Простое и элегантное

Итеративное с очередью (BFS подход):
Временная сложность: O(N)
Пространственная сложность: O(N) для очереди
Избегает переполнения стека вызовов

Итеративное со стеком (DFS подход):
Временная сложность: O(N)
Пространственная сложность: O(H) для стека
Аналог рекурсивного решения без рекурсии
