Оптимальное решение (Бинарный поиск)
Объяснение подхода:
Используем классический бинарный поиск
Когда цикл завершается, left указывает на позицию, куда можно вставить target
Временная сложность: O(log n)
Пространственная сложность: O(1)


Сравнение решений
Метод	                        Время	    Память	Преимущества
Бинарный поиск (итеративный)	O(log n)	O(1)	Оптимальное решение
Бинарный поиск (рекурсивный)	O(log n)	O(log n)Чистая реализация
Линейный поиск	                O(n)	    O(1)	Простота, но не эффективно
