Сравнение решений
Рекурсивное (оптимальное для понимания):
Временная сложность: O(N)
Пространственная сложность: O(H) для стека вызовов (H - высота дерева)
Простое и понятное

Итеративное со стеком:
Временная сложность: O(N)
Пространственная сложность: O(H) для стека
Избегает переполнения стека вызовов для больших деревьев

Morris Traversal:
Временная сложность: O(N)
Пространственная сложность: O(1) - не использует дополнительную память
Сложнее для понимания, модифицирует дерево временно
