Что такое Inorder Traversal?
Inorder (симметричный обход) - это способ обхода бинарного дерева, при котором:
Сначала посещаем все узлы левого поддерева
Затем посещаем корень
Затем посещаем все узлы правого поддерева
Для бинарного дерева поиска (BST) inorder обход возвращает значения в отсортированном порядке.
