Решение Проверка является ли goal подстрокой s + s

Основная идея:
Если строка goal является циклическим сдвигом строки s,
то goal обязательно будет подстрокой строки s + s.

Почему это работает:
Для строки s = "abcde":

s + s = "abcdeabcde"

Все возможные циклические сдвиги содержатся в этой удвоенной строке:
Сдвиг 0: "abcde" (позиции 0-4)
Сдвиг 1: "bcdea" (позиции 1-5)
Сдвиг 2: "cdeab" (позиции 2-6)
Сдвиг 3: "deabc" (позиции 3-7)
Сдвиг 4: "eabcd" (позиции 4-8)

Почему это эффективно:
Сложность по времени: O(n) в среднем случае, O(n²)
в худшем случае (зависит от реализации поиска подстроки)
Сложность по памяти: O(n) для создания строки s + s
