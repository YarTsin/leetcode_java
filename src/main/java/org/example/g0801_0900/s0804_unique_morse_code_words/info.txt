Вариант подхода к решению
Создаем массив с кодами Морзе для всех букв от 'a' до 'z'
Для каждого слова в массиве:
Для каждой буквы в слове получаем соответствующий код Морзе
Конкатенируем все коды в одну строку
Добавляем полученную строку в множество (Set) для
автоматического устранения дубликатов
Возвращаем размер множества

Преимущества:
Использование HashSet обеспечивает автоматическое удаление дубликатов
Временная сложность: O(n * m), где n - количество слов, m - средняя длина слова
Пространственная сложность: O(n * m) для хранения уникальных преобразований
