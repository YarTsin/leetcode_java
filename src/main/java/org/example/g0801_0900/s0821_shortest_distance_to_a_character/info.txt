Один из вариантов решения - Двухпроходный алгоритм
Первый проход (слева направо): Для каждого символа записываем
расстояние до ближайшего c слева

Второй проход (справа налево): Для каждого символа
сравниваем расстояние до ближайшего c справа с уже записанным
расстоянием слева и берем минимум

Математическое обоснование:
Двухпроходный алгоритм работает потому, что для каждой позиции i:
После первого прохода знаем расстояние до ближайшего c слева
После второго прохода знаем расстояние до ближайшего c справа
Минимум этих двух расстояний дает искомый результат
Это решение эффективно обрабатывает все возможные случаи
расположения символов c в строке.
