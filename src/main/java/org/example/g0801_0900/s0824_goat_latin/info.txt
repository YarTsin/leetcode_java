Один из подходов к решению

Подход:
Разбиваем строку на слова
Для каждого слова применяем правила преобразования
Добавляем соответствующее количество букв 'a' в зависимости от позиции слова
Собираем преобразованные слова обратно в строку

Преимущества:
Четкое разделение логики
Легко читается и понимается
Эффективно для данных ограничений
