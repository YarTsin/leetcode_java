Вариант подхода к решению
Инициализируем счетчик строк (lines = 1) и текущую ширину
текущей строки (currentWidth = 0)

Для каждого символа в строке s:
Определяем ширину текущего символа через массив widths
Если добавление ширины символа к currentWidth превышает 100:
- Увеличиваем счетчик строк
- Сбрасываем currentWidth до ширины текущего символа
(переносим символ на новую строку)
Иначе:
- Добавляем ширину символа к currentWidth
Возвращаем массив [lines, currentWidth]
