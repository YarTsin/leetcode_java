Один из вариантов подхода к решению

Нормализация текста: преобразуем весь текст в нижний
регистр и удаляем знаки препинания
Разбиваем текст на слова
Создаем множество запрещенных слов для быстрой проверки
Используем HashMap для подсчета частоты слов
Ищем слово с максимальной частотой, которое не входит в banned

Преимущества:
Временная сложность: O(n + m), где n - длина paragraph, m - количество banned слов
Пространственная сложность: O(n + m)
Эффективная обработка с использованием HashSet и HashMap
