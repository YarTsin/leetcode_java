Один из подходов к решению

Алгоритм:
Инициализация: Создаем пустой список для результатов
Проверка граничных условий: Если строка слишком короткая,
сразу возвращаем пустой результат
Итерация по строке: Проходим по каждому символу,
отслеживая начало текущей группы
Обнаружение конца группы: Когда находим символ,
отличный от следующего, или достигаем конца строки
Проверка длины группы: Вычисляем длину текущей группы
и проверяем, >= 3 ли она
Добавление результата: Если группа большая, добавляем
ее начальный и конечный индексы
Обновление указателя: Перемещаем указатель начала на следующую позицию
