Решение - с использованием счетчика глубины

Подход:
Мы можем отслеживать текущую глубину вложенности папок
относительно корневой директории. Для каждой операции:
"../": уменьшаем глубину на 1 (но не меньше 0)
"./": не меняем глубину
"x/": увеличиваем глубину на 1

После обработки всех операций, текущая глубина покажет,
насколько мы удалились от корня. Именно столько операций
"../" нам понадобится для возврата.
