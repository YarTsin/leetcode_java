Решение - Математическая формула

Подход:
Вычисление длины интервала: length = high - low + 1

Анализ границ:
Если обе границы четные: количество нечетных = length / 2
Во всех остальных случаях: количество нечетных = (length + 1) / 2

Упрощенная формула:
((high - low + 1) + 1) / 2 если low нечетное, иначе (high - low + 1) / 2

Еще более простая формула:
(high + 1) / 2 - low / 2

Обоснование:
Количество нечетных чисел от 0 до n: (n + 1) / 2
Количество нечетных чисел от 0 до high: (high + 1) / 2
Количество нечетных чисел от 0 до low-1: low / 2
Разность дает искомый результат
